Navigation

The navigation consists of a menu used to navigate through information and resources on the portfolo. It helps users find the content they are seeking, and are simple and intuitive.

Accessibility

Ensure that links are navigatable when using screen readers and ARIA. This can be done typically by conveying the menu structure using a list.

Web Version

This navigation is displayed only on browsers. See page patterns for examples.
